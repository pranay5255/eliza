# Eliza AI Trading Platform User Flow

## 1. User Registration & Onboarding

### 1.1 Initial Setup
1. User connects wallet
2. Acquires TRUST governance tokens
3. Registers for platform access

### 1.2 Agent Selection
1. Browse available AI agents (AI Marc, DegenAI)
2. View agent performance metrics and trust scores
3. Select preferred agent(s) for trading

## 2. Trading Flow

### 2.1 Virtual Trading (Phase 1)
1. User deposits virtual assets
2. Selects trading parameters
   - Risk tolerance
   - Investment size
   - Trading pairs
3. Activates AI agent(s)
4. Monitors virtual performance

### 2.2 Paper Trading (Phase 2)
1. Real market data
2. Virtual order execution
3. Performance tracking
4. Risk assessment

### 2.3 Live Trading (Phase 3)
1. Real asset integration
2. Jupiter DEX integration
3. Automated trading execution
4. Real-time monitoring

## 3. Trust Score System
